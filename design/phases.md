## v0 → v0.5 Delta: Build Comparison

- Enable "Compare Build" button in ActionBar
- Load a second build (from URL or localStorage slot)
- Split-panel layout: current build left, comparison right
- Delta indicators: green/red highlights on skill nodes that differ
- Stat radar overlay: both builds on one chart

**Prerequisite:** URL scheme must be active (v0 already stubs this).

---

## v0.5 → v1 Delta: Sharing MVP

- Activate "Share Build" button → copies URL to clipboard, shows confirmation toast
- URL decoding: `?build=<base64>` loads the encoded build on page load
- Spirit Guide: choose and implement placement (sidebar or modal — decision pending)
- Class theming: apply color palette from class JSON to UI accents
- Transition from static class selector to season-aware class picker

---

## v1 → v1.5 Delta: Community Index

- Build a static `build-index.json` (generated by a script, not a live API)
- "Find similar builds" panel on Page 1: query by archetype + element + top skills
- Archetype filter on Page 2 analytics

---

## v1.5 → v2 Delta: Doppelganger Search

- Body-swap orientation flow: "I just swapped into a [class]" → show existing builds for that class
- Post-swap build suggestion: "People who were in this class invested SP like this..."
- Real-time query (requires backend or serverless function)

---

## What Stays Deferred Indefinitely

- Multiplayer / live session sync
- Account system / login
- Gear balance analytics (gear JSON exists but is not consumed)
- Trial data visualization (fights.jsonl)
- Mobile app (PWA shell is feasible at v1.5)
